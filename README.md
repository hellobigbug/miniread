# MiniRead - Windows阅读工具

## 简介

MiniRead 是一款专为 Windows 操作系统设计的阅读工具，采用悬浮置顶的单行滚动显示模式，让您可以在进行其他工作的同时轻松阅读文档内容。

## 系统要求

- **操作系统**: Windows 10 及以上版本
- **Python**: 3.8 或更高版本
- **内存**: 建议 4GB 以上
- **磁盘空间**: 约 50MB

## 安装步骤

### 1. 安装 Python

如果您的系统尚未安装 Python，请从 [Python 官网](https://www.python.org/downloads/) 下载并安装 Python 3.8 或更高版本。

安装时请勾选 "Add Python to PATH" 选项。

### 2. 安装依赖

打开命令提示符（CMD）或 PowerShell，进入项目目录，执行以下命令：

```bash
pip install -r requirements.txt
```

### 3. 运行程序

```bash
python main.py
```

## 功能特点

### 🪟 悬浮置顶窗口

- 窗口始终显示在所有应用程序之上
- 支持半透明背景，不遮挡底层内容
- 可通过鼠标拖动调整窗口位置
- 支持一键隐藏/显示

### 📜 单行滚动显示

- 文本以单行形式滚动显示
- 可自定义滚动速度（10-500 像素/秒）
- 支持向左或向右滚动
- 鼠标悬停时自动暂停滚动

### 🔤 字体自定义

- 支持选择系统已安装的任意字体
- 字号范围：8-72 像素
- 支持粗体和斜体样式
- 可自定义文字颜色

### 📁 多格式支持

- **TXT**: 纯文本文件，自动检测编码
- **PDF**: PDF 文档
- **DOCX**: Microsoft Word 文档
- **EPUB**: 电子书格式
- **HTML/HTM**: 网页文件
- **MD**: Markdown 文件

### ⌨️ 快捷键支持

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+R` | 显示/隐藏窗口 |
| `Ctrl+Shift+Space` | 开始/暂停滚动 |
| `Ctrl+Shift+O` | 打开文件 |
| `Ctrl+Shift+↑` | 增大字号 |
| `Ctrl+Shift+↓` | 减小字号 |
| `Ctrl+Shift+→` | 加快滚动速度 |
| `Ctrl+Shift+←` | 减慢滚动速度 |

### 💾 配置自动保存

- 自动保存窗口位置和大小
- 记住字体偏好设置
- 保存滚动速度和方向
- 记录最近打开的文件
- 保存阅读进度位置

## 使用指南

### 基本使用流程

1. **启动程序**: 运行 `python main.py`
2. **打开文件**: 点击 📁 按钮或按 `Ctrl+Shift+O`
3. **开始阅读**: 点击 ▶ 按钮或按 `Ctrl+Shift+Space`
4. **调整设置**: 根据需要调整字体和滚动速度

### 调整窗口位置

- 将鼠标移动到窗口上
- 按住鼠标左键拖动
- 释放鼠标完成定位

### 字体设置

1. 点击 **A** 按钮打开字体设置对话框
2. 选择喜欢的字体
3. 调整字号（使用滑块或直接输入数值）
4. 可选择粗体或斜体样式
5. 点击颜色按钮选择文字颜色
6. 预览效果后点击"应用"

### 滚动设置

1. 点击 **⚡** 按钮打开滚动设置对话框
2. 调整滚动速度（10-500 像素/秒）
3. 选择滚动方向（向左或向右）
4. 点击"应用"保存设置

### 系统托盘

程序运行时会在系统托盘显示图标：

- **双击图标**: 显示/隐藏窗口
- **右键菜单**:
  - 显示/隐藏
  - 打开文件
  - 退出程序

## 配置文件

配置文件保存在用户目录下：

```
C:\Users\<用户名>\.miniread\config.json
```

### 配置项说明

```json
{
  "window": {
    "x": 100,           // 窗口X坐标
    "y": 100,           // 窗口Y坐标
    "width": 800,       // 窗口宽度
    "height": 60,       // 窗口高度
    "opacity": 0.95     // 透明度 (0-1)
  },
  "font": {
    "family": "Microsoft YaHei",  // 字体名称
    "size": 16,                   // 字号
    "bold": false,                // 粗体
    "italic": false,              // 斜体
    "color": "#FFFFFF"            // 颜色
  },
  "scroll": {
    "speed": 50,              // 滚动速度
    "direction": "left",      // 滚动方向
    "pause_on_hover": true    // 悬停暂停
  }
}
```

## 常见问题

### Q: 程序无法启动？

**A**: 请确保：
1. Python 版本 >= 3.8
2. 已正确安装所有依赖：`pip install -r requirements.txt`
3. 以管理员权限运行（快捷键功能需要）

### Q: 快捷键不起作用？

**A**:
1. 确保安装了 `keyboard` 模块
2. 尝试以管理员权限运行程序
3. 检查是否与其他程序的快捷键冲突

### Q: 文件打开后显示乱码？

**A**:
- 程序会自动检测文件编码
- 如果仍有问题，请尝试将文件转换为 UTF-8 编码

### Q: PDF 文件内容提取不完整？

**A**:
- 部分 PDF 文件（如扫描版）可能无法正确提取文本
- 建议使用文字版 PDF 或转换为其他格式

### Q: 如何重置所有设置？

**A**: 删除配置文件夹：
```
rmdir /s "%USERPROFILE%\.miniread"
```
